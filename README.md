# Descarte-Integrador
Projeto da Disciplina de Projeto Integrador do curso de Ciência da Computação da Universidade Federal do Rio Grande do Sul. O projeto envolve a criação de um aplicativo que peritirá pesquisar e localizar centros de coleta e/ou reciclagem de dejetos que não possam ser facilmente coletados pela coleta seletiva na cidade de Porto Alegre, RS
